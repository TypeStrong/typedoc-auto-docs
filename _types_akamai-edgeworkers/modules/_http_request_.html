  • httpRequest(url: string, options?: { body?: string; headers?: {}; method?: string; timeout?: number }): Promise<HttpResponse>
    • Defined in index.d.ts:585

    Performs a subrequest, fetching the requested resource asynchronously.

    Parameters

    • url: string

      A String containing the URL to fetch. Can be an absolute or relative URL. Relative URLs will use the parent request as the base URL. Only "http" and "https" are supported for the scheme. Specifying port numbers is not supported.

    • Optional options: { body?: string; headers?: {}; method?: string; timeout?: number }

      May include

      • method The HTTP method to use.
      • headers Request headers to specify.
      • body The request payload.
      • timeout The request timeout, in milliseconds.
