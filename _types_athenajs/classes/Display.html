prepareCanvas

  • prepareCanvas(resources: JSObject[]): void
    • Defined in tmpUZesca/_types_athenajs/node_modules/@types/athenajs/index.d.ts:363

    Prepares the canvas before rendering images.

    Explanation: during development, I noticed that the very first time the ctx.drawImage() was used to draw onto a canvas, it took a very long time, like at least 10ms for a very small 32x32 pixels drawImage.

    Subsequent calls do not have this problem and are instant. Maybe some ColorFormat conversion happens.

    This method makes sure that when the game starts rendering, we don't have any of these delays that can impact gameplay and alter the gameplay experience in a negative way.

    Parameters

    • resources: JSObject[]

    Returns void
