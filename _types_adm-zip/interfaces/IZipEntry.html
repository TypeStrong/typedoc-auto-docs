Interface IZipEntry

The IZipEntry is more than a structure representing the entry inside the zip file. Beside the normal attributes and headers a entry can have, the class contains a reference to the part of the file where the compressed data resides and decompresses it when requested. It also compresses the data and creates the headers required to write in the zip file.
