Interface ConnectionPool

ConnectionPool events 'acquire' - emitted whenever pool.acquire is called 'release' - emitted whenever pool.release is called 'query', query - emitted immediately after .query is called on a connection via pool.query. The argument is a Query object. 'close' - emitted when the connection pool has closed all of it connections after a call to close().
