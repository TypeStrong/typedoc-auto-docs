$onAuth

  • $onAuth(callback: Function, context?: any): Function
    • Defined in tmpUZesca/_types_angularfire/node_modules/@types/angularfire/index.d.ts:410

    Asynchronously fires the provided callback with the current authentication data every time the authentication data changes. It also fires as soon as the authentication data is retrieved from the server.

    Parameters

    • callback: Function

      A callback that fires when the client's authenticate state changes. If authenticated, the callback will be passed an object containing authentication data according to the provider used to authenticate. Otherwise, it will be passed null.

    • Optional context: any

    Returns Function

    A function which can be used to deregister the provided callback.
