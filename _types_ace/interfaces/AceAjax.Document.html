indexToPosition

  • indexToPosition(index: number, startRow: number): Position
    • Defined in index.d.ts:505

    Converts an index position in a document to a {row, column} object. Index refers to the "absolute position" of a character in the document. For example:

    var x = 0; // 10 characters, plus one for newline
var y = -1;

    Here, y is an index 15: 11 characters for the first row, and 5 characters until y in the second.

    Parameters

    • index: number

      An index to convert

    • startRow: number

    Returns Position

insertFullLines

  • insertFullLines(row: number, lines: string[]): void
    • Defined in index.d.ts:409

    Inserts the elements in lines into the document as full lines (does not merge with existing line), starting at the row index given by row. This method also triggers the "change" event.

    Parameters

    • row: number

      The index of the row to insert at

    • lines: string[]

      An array of strings

    Returns void

    Contains the final row and column, like this:

    {row: endRow, column: 0}

    If lines is empty, this function returns an object containing the current row, and column, like this:

    {row: row, column: 0}
