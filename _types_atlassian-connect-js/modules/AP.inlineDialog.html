Namespace inlineDialog

The inline dialog is a wrapper for secondary content/controls to be displayed on user request. Consider this component as displayed in context to the triggering control with the dialog overlaying the page content. An inline dialog should be preferred over a modal dialog when a connection between the action has a clear benefit versus having a lower user focus.

Inline dialogs can be shown via a web item target.
