Namespace events

The Events module provides a mechanism for emitting and receiving events.

A event emitted by emit method will only be received by the modules defined in the same add-on. Public events that emitted by emitPublic are used for cross add-on communication. They can be received by any add-on modules that are currently presented on the page.
