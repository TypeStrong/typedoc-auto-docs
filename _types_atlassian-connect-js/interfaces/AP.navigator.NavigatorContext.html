Optional embeddedContentRender

embeddedContentRender: "current" | "version-at-save"
  • Defined in index.d.ts:1159

Identifies the mode for rendering embedded content in Confluence, such as attachments embedded in a page. This only applies to the contentView target. Valid values are:

  • current (render the embedded content using the latest version)
  • version-at-save (render the embedded content using the version at the time the content was saved)

This parameter is optional and defaults to current.
