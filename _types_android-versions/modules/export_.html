Namespace export=

A module to get Android versions by API level, NDK level, semantic version, or version name.

Versions are referenced from here: https://source.android.com/source/build-numbers.html#platform-code-names-versions-api-levels-and-ndk-releases https://github.com/android/platform_frameworks_base/blob/master/core/java/android/os/Build.java

The version for "Current Development Build" ("CUR_DEVELOPMENT") is not included.
