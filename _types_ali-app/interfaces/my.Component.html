setData

  • setData(data: any, callback?: () => void): void
    • Defined in index.d.ts:3140

    将数据从逻辑层发送到视图层，同时改变对应的 this.data 的值

    1. 直接修改 this.data 而不调用 this.setData 是无法改变页面的状态的，还会造成数据不一致。
    2. 单次设置的数据不能超过1024kB，请尽量避免一次设置过多的数据。
    3. 请不要把 data 中任何一项的 value 设为 undefined ，否则这一项将不被设置并可能遗留一些潜在问题

    Parameters

    • data: any

      object 以 key，value 的形式表示将 this.data 中的 key 对应的值改变成 value

    • Optional callback: () => void

      是一个回调函数，在这次setData对界面渲染完毕后调用

        • (): void

        • Returns void

    Returns void
