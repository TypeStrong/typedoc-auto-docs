Optional includeManifest

includeManifest: string | boolean | string[]
  • Defined in index.d.ts:85

Inserts the manifest javascript as a text property in your assets. Accepts the name or names of your manifest chunk. A manifest is the last CommonChunk that only contains the webpack bootstrap code. This is useful for production use when you want to inline the manifest in your HTML skeleton for long-term caching.

false by default.
